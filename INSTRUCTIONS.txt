# Quick Setup:

    1.) Run either setup(TUI).py or setup(GUI).py
    2.) Follow The Instructions
    3.) Run the output.py file generated by setup


# Setup Instructions:

    Setup Is Done Using Either a Graphical Interface (GUI) or a Terminal Interface (TUI).
    Either Can Be Used And Offer The Same Options.

    GUI Method:

        1.) Run "setup(GUI).py"
        2.) Type The Name of the Module You Want to Add
        3.) Press "Add Module"
            3a.) If You Make a Mistake, Press "Reset List"
        4.) When You've Finished Adding Modules, Press "Write"
        5.) Close Setup


    TUI Method:

        1.) Run "setup(TUI).py"
        2.) Type The Name of the Module You Want to Add
        3.) Press Enter
            3a.) If You Make A Mistake, Type "reset"
        4.) When You've Finished Adding Modules, Type "done"


# Running TidPit:

    Once Setup is Finished, Run the "output.py" File That is Created.
    This Will Launch the TidPit Program

# Changing The Program:

    If You Want To Change What Modules Are Used, Simply Delete "output.py",
    and run Your Desired Setup File Again.
